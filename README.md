# Buildo CRM

> AI-CRM с автоматизацией продаж через Telegram-ботов

Один из 6 продуктов экосистемы **Buildo** — ИИ-агентов для бизнеса.

## Что это

Buildo CRM — это ai-crm с автоматизацией продаж через telegram-ботов.

## Стек

- **Next.js 14** + React 18
- **TypeScript 5.7**
- **Tailwind CSS 3.4**
- **Brand color:** `#3B82F6`

## Локальная разработка

```bash
npm install
npm run dev
# → http://localhost:3000
```

## Деплой

Платформа: **Layero** (auto-deploy из main).
